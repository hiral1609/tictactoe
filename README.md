# tictactoe
Tic-Tac-Toe Game
A classic, simple, and interactive Tic-Tac-Toe game built using HTML, CSS, and vanilla JavaScript.

Features
Two-Player Mode: Playable with two players taking turns as 'X' and 'O'. Player 1 starts as 'O'.
Winner Detection: Automatically detects a winning combination (horizontal, vertical, or diagonal) and displays a congratulatory message.
Draw Detection: Recognizes when the board is full with no winner and declares a draw.
Reset & New Game: Includes a "Reset Game" button to restart the current game, and a "New Game" button when a match ends.
Clean UI: Styled with CSS Flexbox for a neat and responsive design.
Technologies Used
HTML5: For the structure and layout of the game board.
CSS3: For styling, colors, box shadows, and responsive layout.
JavaScript (ES6): For the game logic, winning pattern detection, and DOM manipulation.
How to Play
Open the tictactoe.html file in your web browser.
The game starts with Player O.
Players take turns clicking on any of the empty boxes on the 3x3 grid to make their move.
The first player to align 3 of their marks in a row (horizontally, vertically, or diagonally) wins the game.
If all 9 boxes are filled and no player has 3 marks in a row, the game ends in a draw.
Click the Reset Game button anytime to clear the board, or click New Game on the results screen to play again.
